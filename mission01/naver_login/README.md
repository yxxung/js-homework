## TIL
### 1. button type에 따른 이벤트 처리
- 버튼에 타입을 명시하지 않을 경우 기본적으로 type="submit"이다.
- 이 경우, 사용자 지정 이벤트를 추가하면 작동하지 않는다.
- 다른 이벤트를 발생시키고 싶은 경우, type="button"이어야 한다.
